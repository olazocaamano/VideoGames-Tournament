# Product Backlog - eSports Tournament System

This backlog contains only pending features that still need to be developed.

### PB-07 - Player Registration (Account Creation)

**Related User Story:** US-02  

**Description:**  
New users can create an account to access the platform as players.

**Acceptance Criteria:**
- Registration form is available  
- User can enter:
  - Username / Nickname  
  - Email  
  - Password  
- Data is validated  
- User account is stored in the database  
- User can log in after registration  

**Priority:** High
